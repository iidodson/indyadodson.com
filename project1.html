UserTesting

UserTesting hired me to work on their Asset Toolkit, a code based style guide that helps developers keep component styles consistent across their product.

User Interface Design | User Exeperience

About the Project

As the UserTesting platform grew, some product styles began to look different across various platforms. On the frontend engineering team, the problem I was trying to solve was how to centralize these styles across the product.

My Approach

The Asset Toolkit was the answer. It was code based style guide that represented the code and the results of the code for each item in the UserTesting Design System. My role was to redesign the site and make it developer friendly. Research was key in this project. Using the UserTesting platform, I tested other style guides to understand what developers looked for in a style guide and analyzed how users find information in a style guide. I came up with a few wireframes that represented the most preferred navigation style.

Exploration

I explored various directions for each component based on how easy it was for the user to use the information presented.

Result

This project is live and is used internally at UserTesting. It is still an ongoing project where more and more pieces are being added as they grow their Design System.
